# GithubRepoSearchApp

## 🛠 사용 기술

### Core
- React-Native(v.0.71.3)
- TypeScript(v.4.8.4)

### 상태 관리 
- redux-toolkit

### 네비게이션
- react-navigation(v6)

### 통신
- axios & github REST API

### 스타일
- styled-components


## 🚀 프로젝트 실행

### 패키지 설치
```
$ npm install
$ cd ios && pod install && cd .. 
```

#### 개발환경 실행
```
# ios
$ npx react-native run-ios

# android
$ npx react-native run-android
```
